# Discord ChessBot

Discord ChessBot is a feature-rich Discord bot that allows users to play chess games directly within Discord servers. Challenge friends, join tournaments, earn rewards, and enjoy a seamless chess experience with interactive boards and AI opponents.

## Features
- Play chess games with other Discord users
- Challenge, accept, and manage games with slash commands
- AI opponent for solo play
- Tournaments and leaderboards
- In-game shop and rewards system
- Customizable chess piece themes
- Admin controls for managing games and tournaments

## Getting Started
1. Clone the repository
2. Install dependencies with `npm install`
3. Create a `.env` file and add your Discord bot token
4. Start the bot with `npm start`

## Deployment
Supports deployment on Render and other Node.js hosting platforms. Use environment variables for secrets.
